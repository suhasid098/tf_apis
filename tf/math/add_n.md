<a href="../../tf/math/add_n.md"><code>tf.math.add_n</code></a> performs the same operation as <a href="../../tf/math/accumulate_n.md"><code>tf.math.accumulate_n</code></a>.

This op does not [broadcast](
https://docs.scipy.org/doc/numpy-1.13.0/user/basics.broadcasting.html)
its inputs. If you need broadcasting, use <a href="../../tf/math/add.md"><code>tf.math.add</code></a> (or the `+` operator)
instead.

#### For example:



```
>>> a = tf.constant([[3, 5], [4, 8]])
>>> b = tf.constant([[1, 6], [2, 9]])
>>> tf.math.add_n([a, b, a])
<tf.Tensor: shape=(2, 2), dtype=int32, numpy=
array([[ 7, 16],
       [10, 25]], dtype=int32)>
```
